Usage of the application is as follows:

  • Click on the start button to get started
  • Choose your desired estimation calculator from the dropdown menu
  • For a single chemical partition coefficient estimation click on Single Chemical
  • For a batch of chemicals click on Batch Chemicals
  • If the user has Logkow and Logkaw values, the user shall proceed
  • If the user do not have any values, there is a link to Chemspider on the bottom of the window
  • Chemspider has Logkow and Logkaw values for all existing chemicals
  • There are two text boxes for entering Logkow and Logkaw values
  • The user shall only enter one value per line in Batch of Chemicals
  • The user shall enter all the required fields to calculate chemical partition coefficient
  • The user can download the results for batch of Chemicals in an excel format by clicking on "Generate Excel Sheet"
